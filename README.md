<h1>Customer Purchase Prediction</h1>

<p>
This project focuses on predicting whether a customer will make a purchase
based on historical customer data. Machine learning techniques are used
to analyze customer behavior and generate predictive insights.
</p>

<h2>Project Objectives</h2>
<ul>
    <li>Analyze customer demographic and behavioral data</li>
    <li>Build a machine learning model to predict purchase decisions</li>
    <li>Evaluate model performance using suitable metrics</li>
    <li>Support business decision-making in marketing and sales</li>
</ul>

<h2>Dataset</h2>
<p>The dataset contains customer-related attributes such as:</p>
<ul>
    <li>Age</li>
    <li>Gender</li>
    <li>Annual Income</li>
    <li>Spending Score / Purchase History</li>
    <li>Other behavioral or demographic features</li>
</ul>

<h2>Technologies Used</h2>
<ul>
    <li>Python</li>
    <li>NumPy</li>
    <li>Pandas</li>
    <li>Matplotlib / Seaborn</li>
    <li>Scikit-learn</li>
</ul>

<h2>Machine Learning Models</h2>
<ul>
    <li>Logistic Regression</li>
    <li>Decision Tree</li>
    <li>Random Forest</li>
    <li>Support Vector Machine (SVM)</li>
</ul>

<h2>Project Workflow</h2>
<ol>
    <li>Data loading and preprocessing</li>
    <li>Exploratory Data Analysis (EDA)</li>
    <li>Feature selection and scaling</li>
    <li>Model training</li>
    <li>Model evaluation</li>
    <li>Prediction and analysis</li>
</ol>

<h2>Evaluation Metrics</h2>
<ul>
    <li>Accuracy</li>
    <li>Precision</li>
    <li>Recall</li>
    <li>F1-score</li>
    <li>Confusion Matrix</li>
</ul>
